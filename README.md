# Beeswarm Plot — Employee Salary Distribution

This repository contains a Beeswarm Plot visualization created using RAWGraphs,
showing employee salary distribution across departments and experience levels.

The visualization was created as part of a business analytics task requiring:
- RAWGraphs Beeswarm Plot
- Realistic compensation data
- PNG export (≤512px)
- Professional styling suitable for executive use
